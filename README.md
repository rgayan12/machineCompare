## Installation & Usage
- `Git Clone  -` 
- `composer install`
- `npm install`
- Rename .env.example to .env
- Create a database and provide credentials in the .env
- `php artisan migrate` - This should migrate all the tables to your new DB 
- `php artisan serve`
